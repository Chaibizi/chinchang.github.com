Features

  • Only CSS. No JavaScript!
  • Small file size. Only 1.5KB minified and gzipped!
  • Easy to use. No config required.
  • Works in all modern browsers.

Usage

First get the library from the github repo. Then in your HTML include the library CSS: {% highlight css %}{% endhighlight %}

Now, all you need to do is give your element any position class and tooltip text using the data-hint attribute: {% highlight css %}Hello Sir, hover over me.{% endhighlight %}

Use it with other available modifiers in various combinations. Available modifiers: hint--error, hint--info, hint--warning, hint--success, hint--rounded and hint--bounce.

HINT.css

A tooltip library in CSS

W H A T

Hint.css is a tooltip library which uses only CSS to create simple tooltips.

The library itself is written using SASS and uses the BEM convention for naming.

U S A G E

First get the library from above or github repo. Then in your HTML include the library CSS: {% highlight css %}{% endhighlight %}

Any element in your markup which needs to have a tooltip has to be given atleast 2 classes: hint & any one of hint--top, hint--bottom, hint--left, hint--right to position the tooltip. {% highlight css %}Hello Sir, hover me.{% endhighlight %}

And the tooltip content needs to be given using the data-hint attribute on that element. {% highlight css %}Hello Sir, hover me.{% endhighlight %}

Result: Hello Sir, hover me.

Use it with other available modifiers in various combinations. Available modifiers:

  • hint--error
  • hint--info
  • hint--warning
  • hint--success
  • hint--always
